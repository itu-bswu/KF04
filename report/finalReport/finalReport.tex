\chapter{Preface}
\label{PRE}
This report is the result of a project at the bachelor in Software Development 
on the IT University of Copenhagen from March 2011. The project was given as a
First Semester probject in groups of 4 to 5 randomly picked students. There 
were given 2,5 months of the semester to the project. The time not spent on the 
report was used on coding of the actual program, thorough testing and bugfixing.
